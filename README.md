## Introduction

### 網站理念:

- 整理出台灣原住民的慶典時程與每個部落的文化。
- 提供使用者一個可以發文的平台。

### 網站功能:

- 清楚的時間條。
- 快速查詢相關節慶。
- 每個族群的禁忌查詢。
- 提供原住民宣傳祭典文化的平台。
- 提供使用者留言與 Q&A 平台。
- 提供使用者/原住民登入系統（Google）

### 預期使用者:

- 旅客
- 學生
- 想了解原住民文化的人
- 研究人員

### 如何吸引使用者:

- 使用時間條列式列出台灣原住民的節慶與活動。
- 列出每個族群祭典的禁忌。
- 提供原住民宣傳祭典文化的平台。
- 以圖片的方式去呈現，節省過多的文字敘述。
